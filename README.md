# SQL Sales Analysis Project

## Overview
This project demonstrates SQL skills using a SQLite database to analyze sales data.
The database models customers, products, orders, and order items.

## Tools Used
- SQLite3
- SQL
- GitHub

## Database Schema
- customers
- products
- orders
- order_items

## Example Queries
- Total sales per customer
- Best-selling products
- Monthly revenue analysis

## How to Run
```bash
sqlite3 sales.db < schema.sql
sqlite3 sales.db < data.sql
sqlite3 sales.db < queries.sql
